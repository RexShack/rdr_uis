## Red Dead Redemption 2 native menu for RedM.

I advise you not to use it for your server, it is not ready for deployment.

Feel free to contribute for experimented.

## FEATURES:
### Menu
* Open
* Close
* SetHeader
* SetFooter
* SetFooterColor
* AddItem
* RemoveItem
* GetItem
* GetItems

### Menu Item
* SetText
* SetEnabled
* SetVisible
* SetTextColor
